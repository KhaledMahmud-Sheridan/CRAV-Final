
## Use Express Generator

```sh
npx express-generator
npm install
```

### Use this command in windows cmd or powershell to start the site
```sh
$ENV:DEBUG = "nodeapiserver:*"; npm start
```

### Install other required libraries
```sh
npm install mongoose
npm install acync
npm instll express-validator
nom install luxon
npm install express-validator

```
### Install nodemon for auto restart of the server
```sh
npm install nodemon --save-dev
```
### Use the following command in Windows to start nodemon
```sh
node ./node_modules/nodemon/bin/nodemon.js ./bin/www
```
